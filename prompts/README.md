# 服装电商AI图生成 — 提示词工作流

> 由 meta-prompt 系统自动生成
> 输入：「我想用ai生成服装行业的电商图 一套图大概15张」

## 使用流程

```
第1步                    第2步                       第3步
┌─────────────┐    ┌──────────────────┐    ┌─────────────────┐
│  需求采集    │───→│  图组规划         │───→│  质量审查        │
│  (prompt_1)  │    │  (prompt_2)      │    │  (prompt_3)     │
└─────────────┘    └──────────────────┘    └─────────────────┘
     │                    │                       │
     ▼                    ▼                       ▼
 《项目定义卡》     《图组规划方案书》       《质量审查报告》
```

### 第1步：需求采集（prompt_1.md）

1. 打开任意 AI 对话（ChatGPT / Claude / 等）
2. 把 `prompt_1.md` 的全部内容复制粘贴发送
3. AI 会输出一份问卷，你逐项回答
4. AI 整理后输出《项目定义卡》
5. **保存《项目定义卡》，下一步要用**

### 第2步：图组规划（prompt_2.md）

1. 开一个新对话
2. 把 `prompt_2.md` 的全部内容复制粘贴
3. 在后面追加上一步得到的《项目定义卡》全文
4. AI 输出《图组规划方案书》（15张图逐张规划）
5. **保存《图组规划方案书》，下一步要用**

### 第3步：质量审查（prompt_3.md）

1. 开一个新对话
2. 把 `prompt_3.md` 的全部内容复制粘贴
3. 在后面追加《项目定义卡》和《图组规划方案书》全文
4. AI 输出《质量审查报告》
5. 如果审查结果是"需修订"，根据建议修改方案后重新审查
